# 命令参考

Copilot Shell 提供丰富的 slash 命令来控制会话行为。在会话中输入 `/help`
可查看所有可用命令。

## 基础命令

| 命令 | 说明 |
|------|------|
| `/help` | 显示帮助信息 |
| `/clear` | 清屏（快捷键 `Ctrl+L`） |
| `/quit` | 退出 Copilot Shell（别名 `/exit`） |
| `/about` | 显示版本和系统信息 |

## 认证与模型

| 命令 | 说明 |
|------|------|
| `/auth` | 切换认证方式 |
| `/model` | 切换当前使用的模型 |

## 语言与外观

| 命令 | 说明 |
|------|------|
| `/language` | 查看语言设置 |
| `/language ui <lang>` | 设置界面语言（如 `zh-CN`、`en`） |
| `/language output <lang>` | 设置 LLM 输出语言 |
| `/theme` | 切换颜色主题 |
| `/statusline` | 配置状态栏显示 |

## 会话管理

| 命令 | 说明 |
|------|------|
| `/resume` | 恢复历史会话 |
| `/rename` | 重命名当前会话 |
| `/export` | 导出当前会话 |
| `/restore` | 从备份恢复会话 |
| `/compress` | 用摘要替换聊天历史以节省 token |
| `/summary` | 生成当前会话摘要 |
| `/copy` | 复制最近的回复到剪贴板 |
| `/stats` | 显示当前会话的 token 统计信息 |

## 交互式工具

| 命令 | 说明 |
|------|------|
| `/bash` | 进入交互式 shell，输入 `exit` 返回 |
| `/editor` | 打开编辑器编辑内容 |
| `/vim` | 切换 Vim 模式 |
| `/directory` | 浏览目录结构 |

## 工具与权限

| 命令 | 说明 |
|------|------|
| `/tools` | 列出当前可用的工具 |
| `/approval-mode` | 设置工具审批模式 |
| `/permissions` | 管理工具权限 |

审批模式选项：

- `plan`：只规划不执行
- `default`：执行前逐一确认
- `auto-edit`：自动批准文件编辑，其余需确认
- `yolo`：自动批准所有操作（谨慎使用）

## Hooks 管理

| 命令 | 说明 |
|------|------|
| `/hooks` | 查看 hooks 帮助 |
| `/hooks list` | 列出所有已注册 hooks 及状态 |

## 扩展管理

| 命令 | 说明 |
|------|------|
| `/extensions` | 查看已加载的扩展列表 |

## 技能管理

| `/clawhub` | 管理 Clawhub 远程技能 |
| `/agents` | 管理子代理 |

## MCP 服务器

## IDE 集成

## 设置

## Git 与项目

## 其他

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+L` | 清屏 |
| `Ctrl+C` | 取消当前操作 |
| `Ctrl+O` | 切换紧凑模式（隐藏工具输出） |
| `?` | 查看所有快捷键 |
| `Tab` | 命令补全 |
| `↑` / `↓` | 浏览命令历史 |
| `/` | 触发命令列表 |
